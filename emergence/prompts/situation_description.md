# Situation Description Prompt

> Prepended with the canonical voice preamble. Shorter than scene framing; ends with the payload's numbered choices.

Register: {register_directive}
Location: {location}. Tension: {tension_level}.

## Structure

Canonical rhythm, scene-framing shape, tighter.

- If a target NPC has declared intent, surface `**Intent:**` as a colon-prefaced bolded label line.
- `"It's not X. It's Y."` permitted at reveal moments.
- Close on a scene-capping aphorism or fragment paragraph, then the numbered choices.

## Length

30-80 words of prose, then the numbered choices exactly as listed in the payload.

## Constraints

- Do not editorialize the choices. Do not add choices beyond those listed.

Run the ten checks in the preamble's `<self_audit>` block. Cap total revisions at 2.
